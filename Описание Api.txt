	Общее описание Api
Api позволяет возвращать bounding box объектов на изображении


	Описание EndPoints

1)POST http://127.0.0.1:5000/model/?url=<http or local file>
Первый Endpoint используется для обработки изображения через модель передавая через url на изображение в интернете либо на локальном устройстве. Позволяет получить JSON файл на выходе с информацией о bounding box на изображении.
{ "status" : "OK",
  "process_image" : ссылка на http или адрес на файл на локал устройстве (str),
  "object_detected" : [ 
          "label": Название объекта (str),
          "confidence": Точность предсказания (int),
          "bounding_box": 4 координаты чтоб нарисовать прямоугольник (list)
         ]
}


2)GET http://127.0.0.1:5000/model/
Второй Endpoint используется для вывода названия модели которая используется
{ "model_name" : название используемой модели }
